One Extra Step

For the extension to open links in the Firefox browser, please download, unzip and install the open-source native client.

  • Select native runtime:
  • Click here to download the package for you OS (Window, Mac, and Linux operating systems are currently supported) or click NodeJS Runtime or BunJS Runtime to browse all available download options.
  • Extract the downloaded package in a local directory
  • Now open a terminal in this directory (the directory that contains install.sh and uninstall.sh) and run ./install.sh
  • Now double-click the install.bat file

"com.add0n.node" is based on NodeJS and is available for Windows, macOS, and Linux. This package includes NodeJS binary for your operating system and uses it if you don't have NodeJS installed on your system.

"org.webextension.bun" is based on BunJS and is available for macOS, and Linux. For this native client to work, you need to have BunJS installed on your system. The package does not include the native binaries.

The installer script copies a few files to two different locations for your browser to be able to detect this native client. If you already have NodeJS in your system, the installed version of NodeJS will be used. So please make sure you have an up-to-date version of NodeJS installed. If you don't have this application, the portable version will be used. Read more here.

To make sure the native client is accessible, click on the "Check Connection" button. If you still get this page even when the "Check Button" reports successful connection, it means the application cannot be accessed (path is not correct in the options page). If the path to the executable is correct and the native client is connected, you can debug the connection by enabling console logs in Chrome, Edge, Opera, or Firefox.

To completely remove this native client, run ./uninstall.shuninstall.bat.

If you are using an unpacked extension, make sure to include the extension id (-) into the config.js file and reinstall.

Firefox's Command Options
Command Result
-browser Start with the browser component. Firefox and SeaMonkey only.
-foreground Make this instance the active application.
-headless Runs Firefox in headless mode, which is very useful for purposes such as debugging and automated testing.
-new-window URL Open URL in a new window. Firefox and SeaMonkey2.x only.
-private Opens Firefox in permanent private browsing mode. Firefox 3.6 and later only.
-search term Search term with your default search engine. Firefox and SeaMonkey 2.1 and later only.
-url URL Open URL in a new tab or window, depend on the browser option.
